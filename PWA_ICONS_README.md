# PWA Icons

Все иконки для PWA автоматически генерируются из исходного SVG файла.

## Сгенерированные иконки:

- **pwa-192x192.png** - 192x192 пикселей (для Android)
- **pwa-512x512.png** - 512x512 пикселей (для Android)
- **pwa-512x512-maskable.png** - 512x512 пикселей (маскируемая иконка для Android)
- **apple-touch-icon.png** - 180x180 пикселей (для iOS)
- **favicon.png** - 512x512 пикселей (общий favicon)
- **favicon.ico** - 32x32 пикселей (традиционный favicon)
- **mask-icon.svg** - SVG иконка для Safari (macOS/iOS)

## Регенерация иконок:

Если вы изменили `pwa-icon.svg` и хотите пересоздать все PNG иконки:

```bash
npm run generate-icons
```

Этот скрипт автоматически создаст все необходимые размеры иконок из исходного SVG файла.

## Дизайн иконки:

Иконка использует градиентный фон (#667eea → #764ba2) с символом умножения "×" в центре, что соответствует цветовой схеме приложения.

## Размещение файлов:

Все иконки находятся в папке `public/` и автоматически копируются в `dist/` при сборке проекта.
