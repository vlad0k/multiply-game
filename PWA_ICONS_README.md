# PWA Icons Instructions

Для полноценной работы PWA необходимо создать следующие PNG иконки:

## Требуемые иконки:

1. **pwa-192x192.png** - 192x192 пикселей
2. **pwa-512x512.png** - 512x512 пикселей
3. **apple-touch-icon.png** - 180x180 пикселей (для iOS)

## Как создать иконки:

1. Используйте онлайн-генератор: https://realfavicongenerator.net/
2. Или используйте графический редактор (Photoshop, GIMP, Figma)
3. Или конвертируйте SVG файл `pwa-icon.svg` в PNG нужных размеров

## Рекомендации по дизайну:

- Иконка должна быть простой и узнаваемой
- Используйте контрастные цвета
- Убедитесь, что иконка хорошо смотрится на светлом и тёмном фоне
- Для iOS иконка будет автоматически закруглена системой

## Размещение файлов:

Все иконки должны быть размещены в папке `public/`:
- `public/pwa-192x192.png`
- `public/pwa-512x512.png`
- `public/apple-touch-icon.png`

После создания иконок, пересоберите проект:
```bash
npm run build
```
